# nedb
고양이가 귀여운 파일 베이스의 데이터베이스
DB에 값을 쓰면 Json 형식으로 저장된다.

[공식 Github 주소](https://github.com/louischatriot/nedb)

## 설치법

nedb를 설치할 폴더에서 CMD창에 다음 명령어 입력

```npm
npm install nedb --save
```
